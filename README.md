# 🏋️ Тренажёр по русскому языку

Веб-приложение для подготовки к вступительному экзамену по русскому языку в консерваторию.  
Содержит более 300 тестовых заданий с автоматической проверкой ответов и отслеживанием прогресса.

## 🎯 Возможности

- **Два режима работы:**
  - **Тренажёр** – все вопросы в случайном порядке, счётчик пройденных заданий.
  - **Экзамен** – 20 случайных вопросов, сформированных по прототипам реального экзамена.
- **Типы заданий:**
  - Множественный выбор (отметить несколько правильных вариантов).
  - Установление соответствия (сопоставить элементы левого и правого столбцов).
- **Мгновенная проверка** с подсветкой правильных/неправильных ответов.
- **Сохранение прогресса** в `localStorage` (не требует регистрации).
- **История результатов экзамена** с датой и процентом правильных ответов.
- **Адаптивный дизайн** – удобно пользоваться как на компьютере, так и на телефоне.
- **Тёмная тема** в стиле Catppuccin Mocha.

## ⚙️ Технологический стек

- React 18
- TypeScript
- Vite
- CSS Modules
- GitHub Actions (автодеплой на GitHub Pages)

## 🚀 Установка и локальный запуск

```bash
# Клонировать репозиторий
git clone https://github.com/Mudwin/russian-quiz.git
cd russian-quiz

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

Приложение откроется по адресу `http://localhost:5173`.

## 📦 Деплой на GitHub Pages

Проект автоматически деплоится при пуше в ветку `main` с помощью GitHub Actions.
Конфигурация находится в `.github/workflows/deploy.yml`.

Для ручного деплоя:

```bash
npm run predeploy
npm run deploy
```

## 🏗️ Структура проекта

```
src/
  app/                 – главный компонент и его стили
  components/          – переиспользуемые компоненты
    Matching/
    MultipleChoice/
    QuestionRenderer/
    TestArea/
  hooks/               – кастомные хуки (useQuestions, useExamQuestions)
  types/               – интерфейсы TypeScript
  styles/              – глобальные стили и CSS-переменные
  data/                – JSON с вопросами
```

## 📝 Как пользоваться

1. На главном экране выберите **«Тренажёр»** или **«Экзамен»**.
2. В тренажёре отвечайте на вопросы – после проверки сразу видно правильные варианты.
3. В экзамене вам будет предложено 20 заданий, имитирующих реальный тест.
   После завершения появится результат, который сохранится в историю.
4. Вернуться в меню можно в любой момент по кнопке **«В меню»**.
